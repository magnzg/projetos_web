O elemento table (tabela)

O elemento table (tabela) representa dados tabulares, ou seja, informações que são representadas em linhas e colunas

exemplo

Tabela simles com cabeçalho

Nome Sobrenome
Maria Gonzaga
Alice Ferreira

Elementos da tabela

  • th (table header - cabeçalho da coluna)
  • tr (table row - linha da tabela)
  • td (table data - conteúdo/ valor da célula)
